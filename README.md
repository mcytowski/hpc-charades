# HPC Charades
This repository hosts the list of HPC phrases used for the HPC Charades game, as well as a script to generate printable PDFs. The game was first played at the [Cray User Group](https://cug.org) meeting at the Château de Crémat on 29 April 2026 during the legendary CUG Night Out.
